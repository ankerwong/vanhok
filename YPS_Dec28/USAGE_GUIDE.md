## 📝 模块使用指南

## 1️⃣ Content Worksheet

### 功能说明
- 帮助学生定位文章关键信息
- 分为3个部分：关键信息定位、论点分析、批判性思考

### 操作步骤
1. **阅读问题** - 每个问题都有清晰的编号
2. **输入答案** - 在输入框中填写答案
3. **查看参考答案** - 点击"Show Sample Answer"按钮
4. **对比分析** - 绿色框显示参考答案

### 💡 使用技巧
- 短答题：输入简洁答案（单词/短语）
- 长答题：使用大文本框，写3-4句完整段落
- 教师展示时：逐题讲解，点击显示答案引导讨论

---

## 2️⃣ Interactive Flashcards

### 功能说明
- 每篇文章15-20个精选词汇
- 翻转卡片学习单词释义和用法

### 操作步骤
1. **点击卡片** - 卡片会翻转显示背面
2. **查看信息**：
   - 正面：单词 + 音标 + 文章原句
   - 背面：英文释义 + 例句
3. **切换单词** - 使用左右箭头按钮
4. **标记掌握** - 点击"Mark as Mastered"

### ⌨️ 键盘快捷键
- `←` 左箭头 - 上一张卡片
- `→` 右箭头 - 下一张卡片  
- `空格` - 翻转当前卡片

### 📊 进度追踪
- 顶部进度条显示整体掌握情况
- 数字显示：已掌握 / 总数
- 卡片计数器：当前位置 / 总数

### 💡 使用技巧
- 先看正面，尝试回忆释义
- 翻转验证自己的理解
- 反复练习未掌握的单词
- 注意音标和例句用法

## 3️⃣ Audio Podcast

### 功能说明
- AI生成的专业英文朗读
- 2-3分钟概述文章主要内容
- 帮助学生建立整体理解框架

### 播放控制
```
┌─────────────────────────────────────────┐
│          🎵 图标（动画效果）             │
│                                         │
│            ▶️ 播放按钮                   │
│                                         │
│   ◀════════ 进度条 ═════════▶           │
│   0:00                         2:24     │
│                                         │
│   [0.75x] [1x] [1.25x] [1.5x]          │
│                                         │
│   ⏪ 后退10秒  ⏩ 前进10秒  🔄 重播      │
└─────────────────────────────────────────┘
```

### 操作步骤
1. **点击播放按钮** - ▶️ 变为 ⏸️
2. **调整语速** - 点击速度按钮（0.75x - 1.5x）
3. **拖动进度条** - 跳转到指定位置
4. **快退/快进** - 10秒精确控制
5. **重新播放** - 一键回到开头

### 💡 使用技巧
- **初次听**: 使用0.75x或1x，理解大意
- **跟读练习**: 使用0.75x，暂停跟读
- **挑战听力**: 使用1.25x或1.5x
- **重点段落**: 使用快退按钮反复听

### 🎧 教学建议
- 课前播放：激活背景知识
- 课中播放：验证理解
- 课后播放：复习巩固

## 🎓 课堂使用场景

### 场景1: 45分钟标准课程

**时间分配**:
```
00:00-05:00  🎧 Podcast     │ 音频导入，激活背景
05:00-25:00  📝 Worksheet   │ 精读+讨论（主体部分）
25:00-37:00  🎴 Flashcards  │ 词汇学习
37:00-45:00  📚 Background  │ 拓展知识+总结
```

**详细流程**:

1. **开场（5分钟）**
   - 投影展示文章卡片
   - 播放Podcast音频
   - 学生听音频，初步了解主题

2. **精读阶段（20分钟）**
   - 切换到Worksheet模块
   - 学生阅读纸质报刊，完成Part A（10分钟）
   - 教师逐题讲解，点击显示答案（5分钟）
   - 小组讨论Part C开放题（5分钟）

3. **词汇学习（12分钟）**
   - 切换到Flashcards模块
   - 教师演示5个高频词（5分钟）
   - 学生跟读，翻转卡片（5分钟）
   - 快速复习全部单词（2分钟）

4. **拓展总结（8分钟）**
   - 切换到Background模块
   - 展示关键统计数据（3分钟）
   - 连接现实生活案例（3分钟）
   - 课程总结+作业布置（2分钟）

---

### 场景2: 翻转课堂（学生课前预习）

**课前（学生自主）**:
- 🎧 听Podcast（3分钟）
- 📝 完成Worksheet Part A+B（15分钟）
- 🎴 学习Flashcards（10分钟）

**课中（教师引导）**:
- 讨论Worksheet开放题（15分钟）
- 深入分析Background知识（20分钟）
- 小组辩论/角色扮演（10分钟）

---

### 场景3: 自主复习

**学生课后使用**:
1. 重新听Podcast，检验理解
2. 用Flashcards测试词汇记忆
3. 重做Worksheet，对比答案
4. 阅读Background，拓展知识

## 🎨 投影展示技巧

### 最佳显示设置
- **分辨率**: 1920×1080（Full HD）
- **浏览器**: Chrome或Edge（推荐）
- **缩放**: 100%（不要放大）
- **全屏**: 按F11进入全屏模式

### 字体大小
- 标题：1.8-2rem
- 正文：1-1.1rem  
- 说明：0.9rem
- 后排学生清晰可见 ✅

### 颜色对比度
- 深蓝背景 + 白色文字 = 高对比度
- 橙色强调 = 吸引注意
- 绿色答案框 = 正向反馈

## 🐛 常见问题解决

### Q1: 音频无法播放？
**原因**: 音频URL需要特定访问权限
**解决**:
- 检查网络连接
- 刷新页面重试
- 使用本地服务器运行（推荐）

### Q2: 卡片翻转不流畅？
**原因**: 浏览器性能或兼容性
**解决**:
- 使用Chrome/Edge浏览器
- 关闭其他占用资源的程序
- 更新浏览器到最新版本

### Q3: 中文显示为方框？
**原因**: 缺少中文字体
**解决**:
- 确保网络连接正常（加载Google Fonts）
- 等待字体加载完成（首次约3-5秒）

### Q4: 键盘快捷键不响应？
**原因**: 焦点不在Flashcards模块
**解决**:
- 确保当前在Flashcards标签页
- 点击卡片区域获取焦点

## 🖨️ 打印功能

### 打印Worksheet
1. 切换到Worksheet模块
2. 按 `Ctrl+P` （Windows）或 `Cmd+P` （Mac）
3. 选择"打印当前页面"
4. 自动隐藏交互元素，只保留题目

### 打印Flashcards
不建议打印（失去翻转交互效果）
替代方案：导出词汇列表（未来功能）
