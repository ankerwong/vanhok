# ⚡ 快速启动指南

## 🚀 3步开始使用

### 步骤1: 打开网页
```bash
# 方法A: 直接双击
双击 index.html 文件

# 方法B: 使用本地服务器（推荐）
python -m http.server 8000
# 然后访问 http://localhost:8000
```

### 步骤2: 选择文章
- 点击文章卡片（Article 1 或 Article 2）
- 蓝色边框表示已选中

### 步骤3: 开始学习
- 点击顶部导航标签切换模块
- 📝 Worksheet - 完成题目
- 🎴 Flashcards - 学习词汇
- 🎧 Podcast - 听音频
- 📚 Background - 了解背景
