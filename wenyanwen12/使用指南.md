## 💡 核心功能使用說明

### 1. 左右對照閱讀

#### 桌面端使用
```
┌────────────┬────────────┐
│  原文欄    │  譯文欄    │
│  (左側)    │  (右側)    │
└────────────┴────────────┘
```

**使用技巧：**
- 👁️ 眼睛左右來回掃視，對比學習
- 📝 先讀原文，再看譯文確認理解
- 🎯 重點句子多次對照，加深印象

#### 手機端使用
```
┌──────────────┐
│   原文欄      │
│   (上方)      │
├──────────────┤
│   譯文欄      │
│   (下方)      │
└──────────────┘
```

**使用技巧：**
- 📱 上下滑動對照閱讀
- 🔄 可截圖保存重點內容
- 💾 建議橫屏閱讀，空間更大

---

### 2. 智能字詞注釋

#### 如何使用
1. **識別標註字詞**
   - 紫色字體 = 重要字詞
   - 黃色虛線 = 有注釋說明

2. **查看注釋**
   - 將鼠標移到標註字詞上
   - 等待 0.2 秒
   - 自動彈出注釋提示框

3. **閱讀注釋內容**
   - 第一行：字詞釋義
   - 第二行：粵語和普通話讀音（如有）

#### 示例演示

**原文片段：**
```
子曰：「焉得知？」
      ↑
   懸停這裡
```

**顯示效果：**
```
┌─────────────────┐
│  怎麼、哪裡      │
│  粵:jin1|普:yān │
└─────────────────┘
        ▼
     「焉」
```

#### 注意事項
- ⚠️ 手機端：輕觸字詞即可顯示
- ⚠️ 移開鼠標注釋自動消失
- ⚠️ 不阻擋其他文字內容

## 🎯 學習策略建議

### 高效學習法：3-2-1 原則

#### 3 次閱讀
1. **第一次**：快速通讀，了解大意
2. **第二次**：精讀對照，理解細節
3. **第三次**：重點記憶，準備考試

#### 2 種對照
1. **橫向對照**：原文 ↔ 譯文
2. **縱向對照**：本文 ↔ 考點

#### 1 個目標
- 掌握文章內容和考試重點

### 按考頻優先學習

#### 第一優先（★★★★）
- **岳陽樓記**（4次）
  - 重點：憂樂觀、景物描寫
  - 名句：「先天下之憂而憂，後天下之樂而樂」

#### 第二優先（★★★）
- **論語**（3次）
  - 重點：仁孝君子、對比手法
  
- **勸學**（3次）
  - 重點：比喻論證、積累重要
  
- **廉頗藺相如列傳**（3次）
  - 重點：人物刻畫、詳略得宜
  
- **唐詩三首**（4次）
  - 重點：意象分析、情感表達
  
- **詞三首**（4次）
  - 重點：詞風比較、意境營造

#### 第三優先（★★）
