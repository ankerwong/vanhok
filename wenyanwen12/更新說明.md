## ✨ 三大核心改進

### 1️⃣ 原文與譯文左右對照

**之前的設計：**
- 原文和譯文分開在不同區塊
- 需要上下滾動對照閱讀
- 閱讀效率較低

**現在的設計：**
- ✅ 左欄顯示原文，右欄顯示譯文
- ✅ 逐句對照，一目了然
- ✅ 手機端自動切換為上下對照
- ✅ 精美的漸變背景（紫色→黃色）
- ✅ 懸停時整行高亮顯示

**設計特點：**
```
┌─────────────────────┬─────────────────────┐
│   原文（文言文）     │   譯文（白話文）     │
│   襯線字體           │   無襯線字體         │
│   深色文字           │   淺灰文字           │
│   紫色邊框分隔       │                     │
└─────────────────────┴─────────────────────┘
```

---

### 2️⃣ 智能字詞注釋懸停顯示

**之前的設計：**
- 字詞釋義在下方獨立表格中
- 需要滾動到表格查找
- 中斷閱讀流程

**現在的設計：**
- ✅ 重點字詞自動標註（紫色字體 + 黃色虛線）
- ✅ 鼠標懸停即時顯示注釋
- ✅ 紫色漸變提示框，視覺醒目
- ✅ 同時顯示粵語和普通話讀音
- ✅ 無需離開閱讀界面

**示例效果：**
```
原文：子曰：「焉得知？」
      
懸停在「焉」字上時：
┌────────────────┐
│ 怎麼、哪裡        │
│ 粵:jin1 | 普:yān │
└────────────────┘
        ▼
      「焉」
```

**技術實現：**
- HTML5 data 屬性存儲注釋
- JavaScript 動態生成 tooltip
- CSS3 漸變和動畫效果
