# 萬鶴書院 DSE 中文科文言文復習平台 V2.0

## 📚 項目簡介

專為香港 DSE 中文科考生打造的文言文學習平台，提供12篇指定文言經典的深度解析。

### 品牌信息
- **機構名稱**：萬鶴書院 Wanhe Academy
- **品牌色彩**：紫色（#8B4FCF, #9B59D0）+ 金黃色（#F4C542, #FFD700）
- **設計理念**：專業、優雅、學術

## ✨ V2.0 核心功能

### 1. 列表式首頁
- 表格式文章目錄，信息密度更高
- 包含編號、篇目、作者、朝代、考試年份、考頻、查閱按鈕
- 高考頻文章特別標注
- 響應式設計，移動端友好

### 2. 原文譯文對照
- 左右並列式佈局：原文在左，譯文在右
- 桌面端雙欄顯示，移動端上下堆疊
- 紫色漸變分隔線，視覺清晰
- 段落對應，便於對照學習

### 3. 智能字詞注釋
- 懸浮提示（tooltip）功能
- 滑鼠懸停即顯示釋義和讀音
- 紫色字體標注，金黃色虛線下劃線
- 移除獨立詞彙表，注釋即時可見

## 🎯 每篇文章包含內容

### 基本信息
- 作者、朝代
- DSE考試年份
- 考試頻率標注

### 核心內容

2. **智能字詞注釋**（懸浮提示）
3. **主旨分析**
4. **寫作手法**（對比、對偶、借景抒情等）
5. **名句賞析**
6. **DSE考試重點**
   - 高頻考點
   - 常見題型
   - 答題要點
   - 歷年考題解析（如適用）
7. **背景資料**（寫作背景、作者簡介）
