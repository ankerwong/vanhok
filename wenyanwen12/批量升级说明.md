## 📋 升级步骤（针对每篇文章）

### 步骤1：准备内容
1. 整理完整原文（带标点）
2. 准备逐句译文
3. 标记重点字词（需要注释的）
4. 准备字词释义和读音

### 步骤2：HTML结构
```html
<div class="parallel-row">
    <div class="original-column">
        <p>原文，带<span class="annotated" data-meaning="释义" data-pronunciation="读音">标注</span></p>
    </div>
    <div class="translation-column">
        <p>对应的现代汉语译文</p>
    </div>
</div>
```

### 步骤3：添加注释
- 使用 `class="annotated"`
- 添加 `data-meaning="字词解释"`  
- 添加 `data-pronunciation="粵:xxx | 普:xxx"`（可选）

### 步骤4：更新页脚
统一使用：
