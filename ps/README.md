# 万鹤书院 - 个人陈述写作辅导平台

一个专业的个人陈述(Personal Statement)写作辅导网站,帮助中国高中生申请英美澳加香港等地区的世界名校。

![万鹤书院Logo](https://www.genspark.ai/api/files/s/2XpAsX2b)

## 🎯 项目简介

万鹤书院个人陈述辅导平台是由 Anker 老师团队开发的专业升学指导工具,旨在帮助学生:
- 系统学习个人陈述的写作方法
- 借鉴优秀范文的写作技巧
- 使用AI辅助工具快速生成个性化初稿
- 了解不同国家/地区的申请要求

## ✨ 核心功能

### 1. 📚 教学指导课件
横屏PPT式设计,包含8页专业教学内容:
- **课程介绍**: 学习目标和内容概览
- **PS结构**: 开头、主体、结尾的详细解析
- **写作要点**: 5大核心要素(价值观、具体性、深度思考、个人化、逻辑连贯)
- **各地要求**: 英美澳加香港名校的具体要求对比
- **常见错误**: 10大错误及避免方法,附自查清单
- **优秀开头**: 4种有效策略(转折式、坦白式、哲学式、场景式)
- **段落结构**: 80/20黄金比例、蒙太奇vs叙事结构
- **AI使用规范**: ⚠️ 正确使用AI工具,避免学术诚信问题

**特色**:
- 支持键盘操作(←→)和触摸滑动
- 侧边导航快速跳转
- 底部进度指示器

### 3. 🤖 AI个人陈述生成器 ⭐核心功能
智能表单收集学生信息:
- **基本信息**: 姓名、年龄
- **申请信息**: 
  - 目标院校(32所世界名校下拉列表,按地区分组)
  - **自定义院校输入框**(新增!如果列表中没有您的目标院校)
  - 目标专业(16个常见专业)
- **学术背景**: GPA/成绩、获奖情况、学术兴趣点
- **个人特质**: 8种核心特质多选
- **课外活动**: 实习、志愿者、活动经历
- **兴趣爱好**: 个人爱好
- **个人故事**: 重要经历或转折点
- **字数要求**: 500/650/800/1000词可选

**生成逻辑**:
- 4种开头风格随机选择(转折/坦白/哲学/场景)
- 遵循80/20原则(80%学术,20%个人)
- 包含学术段落、经历段落、个人故事、未来规划
- 有力的结尾呼应开头

**⚠️ AI使用警告**:
- 生成器页面顶部有醒目的黄色警告框
- 结果显示区域上方有红色提示
- 教学课件中专门增加"AI使用规范"一页
- 明确说明AI使用风险和正确做法

**AI分析功能**:
- 结构分析
- 核心价值观识别
- 具体改进建议(8-10条)

**操作功能**:
- 在线编辑
- 一键复制
- 下载为TXT文件
- 重新生成

## ⚠️ 重要提醒

### AI工具使用规范

**🚨 本网站的AI生成器仅供学习和参考,切勿直接提交AI生成的内容!**

#### 使用AI的风险:
1. **可能被检测**: 多数顶尖大学使用GPTZero、Turnitin等AI检测工具
2. **缺乏个性**: AI无法真实反映您的独特经历和思考深度
3. **学术诚信**: 部分院校明确禁止使用AI生成申请材料
4. **面试露馅**: 如果PS不是您写的,面试时无法深入阐述

#### 正确使用AI的方式:
1. ✅ 用于头脑风暴和结构建议
2. ✅ 作为语言润色工具
3. ✅ 生成初稿后进行10-20次人工修改
4. ✅ 寻找范文学习写作技巧
5. ❌ 不要直接复制粘贴AI生成内容
6. ❌ 不要让AI编造经历和细节
7. ❌ 不要完全依赖AI而不自己思考

#### 优秀PS的必备要素:
- 真实的个人经历
- 具体的细节和数据
- 深度的反思和洞察
- 独特的个人声音
- 与其他申请材料的一致性

## 🎓 使用建议

### 对学生:
1. 先完整学习教学课件,理解PS写作原理
2. 仔细阅读优秀范文,学习写作技巧
3. 使用AI生成器创建初稿
