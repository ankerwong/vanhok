# 快速开始指南

欢迎使用万鹤书院个人陈述辅导平台!本指南将帮助您快速上手使用本网站。

## 🚀 立即使用

### 方法1: 本地打开(最简单)
1. 确保所有文件都在同一文件夹中
2. 双击 `index.html` 文件
3. 网站将在默认浏览器中打开
4. 开始使用!

### 方法2: Web服务器部署
```bash
# 如果您有Python
cd wanhe-academy
python -m http.server 8000

# 如果您有Node.js
npx serve

# 然后在浏览器访问
# http://localhost:8000
```

### 方法3: GitHub Pages部署
1. 创建GitHub账号(如果还没有)
2. 创建新仓库,上传所有文件
3. 前往 Settings > Pages
4. 选择 main 分支,点击 Save
5. 等待几分钟,访问提供的URL

**提示**:

- 建议完整阅读一遍再进入下一步

### 第二步:借鉴优秀范文

1. 点击导航栏的 **"优秀范文"**
2. 选择您感兴趣的专业类别
3. 点击卡片查看完整范文
4. 重点关注:
   - 开头如何吸引人
   - 如何展现学术深度
   - 个人故事如何融入
   - 结尾如何有力
   - 专家点评的亮点分析

**提示**:
- 8篇范文涵盖8个不同专业
- 每篇都有字数统计和专家点评
- 不要抄袭,而是学习思路和技巧

---

### 第三步:使用AI生成器创建初稿

1. 点击导航栏的 **"AI生成器"**

2. **⚠️ 首先阅读黄色警告框**:
   - 了解AI使用风险
   - 明确这只是初稿
   - 需要大量修改

3. 填写表单:
   - **基本信息**: 姓名、年龄
   - **申请信息**: 
     * 从下拉列表选择目标院校(32所)
     * 或在"其他院校"框中输入自定义院校名称
     * 选择目标专业
   - **学术背景**: GPA、获奖、学术兴趣(尽量详细)
   - **个人特质**: 勾选3-5个核心特质
   - **课外活动**: 详细描述2-3个重要经历
   - **兴趣爱好**: 简单列举
   - **个人故事**: 如有重要转折点,详细描述
   - **字数要求**: 根据目标院校选择

4. 点击 **"生成个人陈述"**

5. 查看生成结果:
   - 阅读生成的PS
   - 查看AI分析和建议
   - **注意红色警告框!**

6. 开始修改:
   - 点击 **"编辑"** 按钮在线修改
   - 或点击 **"下载"** 在本地编辑器修改
   - 点击 **"复制"** 复制到Word/Google Docs

**提示**:
- 输入的信息越详细,生成效果越好
- 重点填写"学术兴趣"和"个人故事"
- 自定义院校输入框支持任何大学名称
- 生成后查看"改进建议"很重要

---

### 第四步:大量修改和完善(最重要!)

**🚨 这是最关键的步骤!**

1. **第一轮修改**(结构和内容):
   - 检查开头是否吸引人
   - 补充具体的数字和细节
   - 删除空洞和笼统的描述
   - 确保80%学术,20%个人

2. **第二轮修改**(个性化):
   - 添加真实的个人经历
   - 融入独特的思考和见解
   - 展现你的个人声音
   - 避免AI的通用语言

3. **第三轮修改**(针对性):
   - 研究目标院校的特色
   - 提及具体的教授、项目或资源
   - 说明为什么选择这所学校
   - 确保与其他申请材料一致

4. **第四轮修改**(语言润色):
   - 检查语法和拼写
   - 优化句子结构
   - 删除重复和啰嗦
   - 确保字数符合要求

5. **第五轮及以后**(反复打磨):
   - 大声朗读,检查流畅度
   - 请老师、同学校对
   - 放置一天后重读
   - 继续修改直到满意

**目标**: 至少修改10-20次!

## 💡 专业建议

### 写作技巧
- ✅ 用具体例子代替笼统描述
- ✅ 展现深度思考,不只是列举成就
- ✅ 保持真实,不编造经历
- ✅ 展现热情,但不浮夸
- ✅ 每段都有明确主题

### 常见错误
- ❌ 陈词滥调的开头("从小我就...")
- ❌ 简历式罗列活动
- ❌ 语言过于正式或做作
- ❌ 缺乏具体性和细节
- ❌ 忽视"为什么"和反思
- ❌ 弱的结尾
- ❌ 语法和拼写错误
- ❌ 所有学校用同一篇

### 自查清单
在提交前,确保:
- [ ] 字数符合要求(不超标也不过短)
- [ ] 开头吸引人,避免陈词滥调
- [ ] 有具体的事例和细节支撑
- [ ] 展现了深度思考和反思
- [ ] 结尾有力,呼应主题
- [ ] 语言自然,有个人风格
- [ ] 无语法和拼写错误
- [ ] 所有经历真实可查
- [ ] 至少3人校对过
- [ ] 针对该校定制,不是通用版本

## ⚠️ 关于AI使用的重要说明

### 风险提醒
1. **检测风险**: 顶尖大学使用AI检测工具,识别率很高
2. **诚信问题**: 部分学校明确禁止AI生成申请材料
3. **缺乏个性**: AI无法展现你的独特性
4. **面试风险**: 如果PS不是你写的,面试时会露馅

### 正确做法
- ✅ 把AI生成的内容当作**灵感和框架**
- ✅ 进行**至少10-20次**大量修改
- ✅ 融入**真实经历和独特思考**
- ✅ 确保最终版本**完全用你自己的语言**
- ✅ 请**多人校对**,确保真实性

### 错误做法
- ❌ 直接复制粘贴AI生成内容
- ❌ 让AI编造经历和细节
- ❌ 完全依赖AI不自己思考
- ❌ 不进行充分修改就提交

**记住**: 招生官想了解的是**真实的你**,不是AI。你的独特经历、思考方式和成长故事才是最有价值的!

## 🆘 常见问题

### Q: 我的目标院校不在列表中怎么办?
A: 在"目标院校"下拉列表下方有"其他院校"输入框,您可以直接输入任何院校的英文名称,如 "ETH Zurich", "Tsinghua University" 等。

### Q: 生成的PS可以直接提交吗?
A: **绝对不可以!** 这只是初稿,必须进行10-20次人工修改,融入真实经历和个人思考。

### Q: 如何让生成的PS更好?
A: 关键是输入详细的信息,特别是"学术兴趣"和"个人故事"部分。输入越具体,生成效果越好。但无论如何,都需要大量修改。

### Q: 字数不够/太多怎么办?
A: 点击"编辑"按钮,手动增删内容。也可以点击"重新生成"试试不同版本。

### Q: 可以生成多个版本吗?
A: 可以!点击"重新生成"按钮会生成新版本(会有一定随机性)。建议生成2-3个版本,取长补短。

### Q: 范文可以直接用吗?
A: **不可以!** 范文只供学习参考,抄袭是严重的学术不诚信行为。学习的是写作思路和技巧,而不是复制内容。

### Q: 不同国家的PS有什么区别?
A: 详见教学课件第4页"各地要求"。简单来说:
- 英国(UCAS): 4000字符,更重学术(80/20)
- 美国(Common App): 650词,更重个人故事
- 香港: 通常1000词左右,平衡学术与个人
- 详细要求请查看各校官网
