万鹤书院合作教师管理制度

致亲爱的合作教师们

欢迎加入万鹤书院的教师团队！作为我们重要的合作伙伴，您的专业能力和敬业精神是我们共同成功的基础。本制度旨在建立良好的合作关系，确保教学质量，维护双方利益。

教师形象与职业素养

  • • 着装得体，颜色不限，注意整洁
  • • 男教师：保持头发和指甲整洁
  • • 女教师：化淡妆，注意仪表管理
  • • 注意个人卫生和体味管理
  • • 佩戴工牌，体现专业形象

教学质量管理

  • • 提前了解学生需求，制定教学计划
  • • 授课前提前5-10分钟到达准备
  • • 教案需提前两天提交审核
  • • 课后48小时内完成反馈填写
  • • 按照统一教学大纲执行

师生沟通规范

  • • 使用企业微信添加学生联系方式
  • • 所有沟通需通过机构指定渠道
  • • 课程调整需通过教务部统一安排
  • • 发现学生其他需求需告知教务
  • • 维护良好的师生关系

机构活动参与

  • • 积极参加教研会议
  • • 参与消防演练等安全活动
  • • 参加年度团建活动
  • • 协助学生活动开展
  • • 配合机构统一安排

保密协议与商业道德

作为合作教师，您需要严格遵守保密义务，保护学生隐私和机构商业机密：

  • • 不得泄露学生个人信息及家庭情况
  • • 保护教学材料和教研资料
  • • 不得转移学生至校外辅导
  • • 维护机构声誉和商业利益
  • • 遵循基本商业道德准则

红线行为（零容忍）

以下行为属于严重违规，将面临严厉处罚：

  • 私自添加学生/家长微信或其他联系方式
  • 引导学生进行私下补课或校外辅导
  • 介绍学生给其他非本机构教师
  • 以"直接找我和去机构一样"为由截留学生
  • 泄露学生个人信息或机构商业机密

红线行为处罚措施：

  • • 第一次发现：警告 + 扣除当月50%课时费
  • • 第二次发现：立即停止合作 + 不发剩余课时费 + 保留追究法律损失权利

普通违规行为处理

对于以下行为，我们将采取相应的教育和提醒措施：

  • • 未及时沟通教学计划及结果
  • • 私自调课（未经教务同意）
  • • 未按时汇报教学内容
  • • 使用未经审核的教学材料

处理方式：第一次口头提醒，第二次书面警告，第三次扣除当月10%课时费

合作共赢理念

我们相信，通过建立清晰的合作规范，能够为教师提供更好的教学环境，为学生提供更优质的教育服务。让我们携手共进，为教育事业贡献我们的力量！
