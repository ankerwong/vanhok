# 万鹤书院门牌生成器（A4 一页双门牌）

## 项目目标
为万鹤书院（Vanhok Academy）提供一个静态网页工具：用户填写两张门牌信息，一键生成 **A4 大小（画布 768×1024px / 3:4）** 的图片，其中包含 **上下两张 A5 门牌**（便于打印后裁切）。

## 已完成功能
- 现代风格 UI（深紫蓝 + 金色，符合品牌调性）
- 表单输入两张门牌信息：
  - 房号（Room Number）
  - 优先使用者（英文）
  - 开始日期、结束日期
- 一键生成 A4 预览图（Canvas 绘制）：
  - 渐变背景 + 几何/六边形纹理 + 玻璃拟态面板
  - 门牌左上角放置 **完整万鹤书院 Logo**
  - 房号金色大字
  - Priority User / Period 信息
- 支持下载：PNG / JPG（JPG 可调质量）
- 响应式布局（桌面/平板可用）
- 基础校验：必填、结束日期需晚于开始日期

## 功能入口（URI）
- `/index.html`：主页面（表单 + 预览 + 下载）

## 未实现功能

- 自定义主题（不同背景纹理、字号、布局模板）
- 批量生成多页 / 多房间导出
