# 萬鶴書院 - HKDSE英語口語考試分組系統

## 📋 系統簡介

這是專為萬鶴書院設計的HKDSE英語口語考試抽籤分組系統，完美適配1920×1080大屏幕顯示。

## ✨ 核心功能

### 1. 學生名單輸入
- 支持批量複製粘貼學生名單
- 每行一個學生姓名
- 實時顯示已輸入學生數量

### 2. 考場設置
- 可選擇2-4個考場同時進行
- 為每個考場指定負責老師姓名
- 動態生成教師輸入框

### 3. 智能分組算法
- **優先4人一組**，最大化4人組數量
- 當學生總數不是4的倍數時，智能處理：
  - 總數除以4餘1：生成(n-1)個4人組 + 2個3人組
  - 總數除以4餘2：生成(n-1)個4人組 + 2個3人組  
  - 總數除以4餘3：生成n個4人組 + 1個3人組
- 隨機打亂學生順序確保公平性
- 輪流分配小組到各個考場

### 4. 結果顯示
- 大屏幕友好設計（1920×1080）
- 清晰展示每個考場的分組情況
- 顯示考場號、負責老師、小組號、學生姓名
- 統計總學生數和總組數

## 🚀 使用方法

### 步驟1：準備學生名單
```
張三
李四
王五
陳六
...
```

### 步驟2：輸入信息
1. 將學生名單複製粘貼到文本框
2. 選擇考場數量（2-4個）
3. 填寫每個考場的負責老師姓名

### 步驟3：開始抽籤
點擊「開始抽籤分組」按鈕，系統自動：
- 隨機打亂學生順序
- 智能分配3-4人小組
- 均衡分配到各考場

### 步驟4：查看結果
在1920×1080大屏幕上清晰顯示：
- 每個考場的分組詳情
- 每組學生名單
- 統計信息

## 📊 分組示例

### 示例1：40名學生，3個考場
- 結果：10個4人組
- 考場1：4組（16人）
- 考場2：3組（12人）
- 考場3：3組（12人）

### 示例2：38名學生，3個考場
- 結果：8個4人組 + 2個3人組
- 考場1：4組
- 考場2：3組
- 考場3：3組

3. 在1920×1080分辨率下顯示效果最佳

## 🔄 重新分組

如需重新分組，點擊「返回重新分組」按鈕即可返回設置頁面，修改信息後重新抽籤。
