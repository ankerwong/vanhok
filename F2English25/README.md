# 万鹤书院DSE - HKDSE中二英语秋季班招生网站

## 项目概述

这是一个专业的HKDSE中二英语秋季班招生网站，为万鹤书院DSE培训机构设计。网站采用现代化响应式设计，提供完整的课程信息、师资介绍、收费标准和报名咨询功能。

## 🎯 项目目标

- 展示HKDSE中二英语秋季班课程信息
- 突出明星教师Tina王老师的专业资质
- 清晰展示课程安排、收费和退费政策
- 提供便捷的咨询报名渠道
- 吸引目标学生和家长报名
