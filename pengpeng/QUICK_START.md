# 🚀 快速上手指南

## 5分钟学会使用面试提词器

### 第一步：打开应用 📱

1. 在浏览器中打开 `index.html` 文件
2. 看到欢迎界面，包含：
   - 顶部：控制按钮
   - 左侧：12个问题列表
   - 中间：内容显示区
   - 底部：进度条

### 第二步：基础操作 🎮

#### 播放控制
- 点击中间的 **▶️ 播放按钮** 开始自动滚动
- 再次点击变为 **⏸ 暂停**
- 或按 **空格键** 快速切换

#### 速度调节
选择三档速度：
- **慢** - 适合初次练习
- **中** - 推荐日常使用
- **快** - 熟练后加速

#### 语言切换
- 点击右上角 **🌐 中文/English** 按钮
- 或按 **←** 切换到中文，**→** 切换到英文

### 第三步：导航功能 🧭

#### 快速跳转
- 点击左侧问题列表中的任意问题
- 内容区会自动滚动到对应位置
- 当前问题会高亮显示

#### 进度显示
- 底部显示当前问题标题
- 进度百分比实时更新
- 进度条可视化显示位置

### 第四步：个性化设置 ⚙️

#### 主题切换
- 点击 **🌙 月亮图标** → 暗色主题（适合夜间）
- 点击 **☀️ 太阳图标** → 亮色主题（适合白天）

#### 字体大小
- 点击 **📏 字体图标** 循环切换
- 正常 → 大 → 超大 → 正常

### 第五步：高级技巧 💡

#### 全屏模式
按 `F` 键进入全屏，专注练习

#### 重置位置
按 `R` 键快速回到顶部

#### 查看快捷键
按 `?` 键显示所有快捷键提示

## 📝 使用场景

### 场景1：初次熟悉内容
1. 选择 **慢速**
2. 使用 **中文** 模式
3. 逐个问题仔细阅读
4. 点击左侧导航跳转复习

### 场景2：中英文对照学习
1. 先读中文版理解内容
2. 按 `→` 切换到英文
3. 对比学习表达方式
4. 记住关键术语

### 场景3：模拟面试练习
1. 选择 **中速**
2. 打开 **全屏模式** (按F)
3. 跟随滚动自然表达
4. 注意眼神和节奏

### 场景4：考前快速复习
1. 选择 **快速**
2. 浏览所有内容
3. 重点问题点击跳转
4. 确认关键要点

## ⚠️ 常见问题

### Q: 滚动太快怎么办？
**A:** 点击暂停按钮，或按↓减速，选择更慢的速度档位。

### Q: 如何只看英文版本？
**A:** 点击右上角语言按钮切换到English，或按→箭头键。

### Q: 可以跳过某些问题吗？
**A:** 可以！点击左侧任意问题直接跳转。

### Q: 暗色模式看不清？
**A:** 点击太阳图标切换回亮色主题，或调大字体。

### Q: 忘记快捷键了？
**A:** 按?键显示快捷键提示，或查看页面上的提示。

## 💪 练习建议

### 第一周：熟悉内容
- 每天使用20分钟
- 完整浏览2-3遍
- 理解每个问题要点
- 标记重点和难点

### 第二周：深化记忆
- 增加到每天30分钟
- 中英文交替练习
- 尝试脱稿复述
- 录制练习视频

### 面试前一天：最后冲刺
- 快速复习全部内容
- 模拟完整面试流程
- 确认关键数据准确
- 放松心态，早点休息

## 🎓 成功关键

1. **理解而非背诵** - 掌握核心思想，用自己的话表达
2. **反复练习** - 熟能生巧，至少练习5次以上
3. **自然表达** - 保持眼神交流，避免机械朗读
4. **灵活应变** - 根据面试官反应调整内容
5. **保持自信** - 相信自己的准备和能力
