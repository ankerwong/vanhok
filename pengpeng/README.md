# CityU Marketing 面试提词器

一个专业的面试提词器网页应用，专为香港城市大学市场营销专业线上面试设计。提供自动滚动、中英文切换、问题导航等实用功能，帮助你在面试中自信流畅地表达。

## 🌟 核心功能

### 📝 内容管理
- ✅ **12个完整问题**：包含自我介绍、专业选择、职业规划等核心面试问题
- ✅ **中英双语**：每个问题都配有完整的中英文版本
- ✅ **一键切换**：快速在中英文之间切换，保持阅读流畅

### 🎮 智能控制
- ✅ **自动滚动**：支持慢/中/快三档速度调节
- ✅ **播放控制**：随时暂停/继续滚动
- ✅ **进度显示**：实时显示阅读进度和当前问题
- ✅ **智能停止**：滚动到底部自动停止

### 🎨 界面设计
- ✅ **亮色/暗色主题**：适应不同环境光线
- ✅ **三级字体大小**：正常/大/超大字体自由切换
- ✅ **响应式布局**：完美支持各种屏幕尺寸
- ✅ **现代化UI**：简洁专业的视觉设计

### ⌨️ 快捷键支持
- `空格键`：暂停/播放
- `↑`：加速滚动
- `↓`：减速滚动
- `←`：切换到中文
- `→`：切换到英文
- `R`：重置到顶部
- `F`：全屏模式
- `?`：显示快捷键提示

### 📱 导航功能
- ✅ **问题列表**：左侧显示全部12个问题
- ✅ **快速跳转**：点击问题快速定位
- ✅ **当前高亮**：自动高亮显示当前正在阅读的问题
- ✅ **滚动同步**：滚动时自动更新当前问题

## 📋 问题列表

本提词器包含以下12个面试问题：

1. **自我介绍** (Self-introduction)
2. **为什么选择香港城市大学Marketing？** (Why CityU Marketing?)
3. **我们为什么要给你发offer？** (Why should we offer you a place?)
4. **你对i-Marketing的理解是什么？** (What is your understanding of i-Marketing?)
5. **未来职业规划** (What are your career goals?)
6. **副学士学习期间的收获是什么？** (What did you gain from your Associate Degree studies?)
7. **某门课程的收获是什么？** (What did you learn from a specific course?)
8. **销售与市场营销的区别是什么？** (What is the difference between sales and marketing?)
9. **你最喜欢的品牌/营销活动/公司是什么？** (What is your favorite brand/marketing campaign/company?)
10. **向面试官提出的问题？** (Do you have any questions for us?)
11. **AI Marketing怎么看？** (What do you think of AI Marketing?)
12. **如何改进商业领袖论坛？** (How would you improve the Business Leaders Forum?)

## 🚀 使用方法

### 基础操作

1. **打开应用**：在浏览器中打开 `index.html`
2. **选择语言**：点击右上角语言按钮切换中英文
3. **开始滚动**：点击中间的播放按钮开始自动滚动
4. **调整速度**：选择慢/中/快三档速度
5. **跳转问题**：点击左侧问题列表快速定位

### 高级功能

- **主题切换**：点击月亮/太阳图标切换亮色/暗色主题
- **字体调整**：点击字体图标循环切换字体大小
- **全屏模式**：按 `F` 键进入全屏专注模式
- **重置位置**：按 `R` 键快速回到顶部

### 面试技巧

1. **提前熟悉**：面试前多次浏览，熟悉内容结构
2. **调整速度**：根据个人阅读速度选择合适档位
3. **标记重点**：记住关键词，不要完全依赖提词器
4. **自然表达**：理解内容后用自己的话表达，避免机械背诵
5. **模拟练习**：使用提词器进行多次模拟面试

## 🔧 自定义配置

### 修改滚动速度

在 `js/main.js` 中修改速度参数：

```javascript
// 慢速：0.5，中速：1，快速：2
function setSpeed(speed) {
    scrollSpeed = speed;
}
```

### 添加新问题

在 `js/data.js` 中添加新的问题对象：

```javascript
{
    id: 13,
    title: "你的新问题",
    titleEn: "Your New Question",
    contentCn: "中文内容...",
    contentEn: "English content..."
}
```

### 修改主题颜色

在 `css/style.css` 中修改CSS变量：

```css
:root {
    --accent-primary: #0d6efd;  /* 主色调 */
    --accent-hover: #0b5ed7;    /* 悬停色 */
}
```

## 💡 使用建议

### 面试前准备
1. **内容理解**：深入理解每个问题的核心要点
2. **个性化调整**：根据自己的经历修改内容
3. **多次练习**：使用提词器进行至少3-5次完整练习
4. **时间把控**：注意每个问题的回答时长

### 面试时使用
1. **设备准备**：提前测试设备和网络连接
2. **位置调整**：将提词器放在摄像头附近，保持自然眼神
3. **速度设置**：选择比练习时稍慢的速度
4. **自然表达**：将提词器作为提示，而非完全照读

### 注意事项
⚠️ **不要完全依赖**：提词器是辅助工具，理解内容最重要  
⚠️ **保持眼神交流**：适当看向摄像头，避免一直盯着屏幕  
⚠️ **自然表达**：用自己的语言和节奏表达，避免机械朗读  
⚠️ **准备Plan B**：万一技术故障，确保能凭记忆回答核心问题
