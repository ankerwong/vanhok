## 🎯 **系統功能**

### 1️⃣ 輸入方式

#### **Excel批量導入** ⭐ 推薦
- 📥 下載Excel模板
- 📊 填寫考生資料（9個字段）
- 📤 上傳Excel文件
- ✅ 自動解析並導入

**Excel模板字段：**
```
1. 姓名（中文）
2. 英文姓名
3. 考生編號
4. 性別（M/F）
5. 科目（中英對照）
6. 應考語言
7. 考場
8. 考試日期（YYYY-MM-DD）
9. 考試時間
```

#### **手動逐一輸入**
- 適合少量考生
- 表單驗證，防止錯誤
- 逐一添加到列表

### 2️⃣ 準考證內容

#### **頂部區域**
```
┌────────────────────────────────────────────┐
│  [Logo]     香港中學文憑考試        [條形碼]│
│          HKDSE Examination                  │
│          准考證 Admission Form              │
└────────────────────────────────────────────┘
```

#### **基本信息**
- 考生姓名（中英文）
- 考生編號（大字顯示）
- 性別（M/F）
- 入境許可證號（XX12345678）

#### **考試詳情表格**
| 科目/卷別 | 應考語言 | 試場名稱及地址 | 試場編號 | 日期 | 時間 |
|----------|---------|--------------|---------|------|------|
| 科目內容 | 語言 | 考場地址 | -- | 日期 | 時間 |

#### **備註區域**
- 中英文對照考試須知
- 保持HKDSE官方格式
- 包含重要提醒事項

### 3️⃣ 自動化功能

**條形碼生成**
- 格式：CODE128
- 基於考生編號自動生成
- 顯示在右上角

**日期格式化**
- 輸入：2025-04-15
- 輸出：15-APR (Mon)

**智能填充**
- 入境許可證號自動填充
- 試場編號留空（--）
- 備註內容自動添加

### 4️⃣ PDF導出

**批量下載功能**
- ✅ 一鍵生成所有考生準考證
- ✅ 合併為單個PDF文件
- ✅ 每頁A4尺寸
- ✅ 適合直接打印
- ✅ 文件命名：`準考證_批量下載.pdf`

**生成過程**
1. 點擊"批量下載PDF"
2. 顯示進度（1/40, 2/40...）
3. HTML → Canvas → PDF
4. 自動下載

## 🚀 **使用流程**

### 快速開始（3步驟）

**步驟1：下載模板**
```
1. 打開index.html
2. 點擊"下載Excel模板"
3. 獲得標準格式模板
```

**步驟2：填寫資料**
```
在Excel中填寫考生信息：
- 姓名：張偉明
- 英文姓名：ZHANG Weiming
- 考生編號：A123456
- 性別：M
- 科目：English Language 英國語文
- 應考語言：英文 English
- 考場：Room 101, Vanhok Academy
- 考試日期：2025-04-15
- 考試時間：09:00 - 11:30
```

**步驟3：生成下載**
```
1. 上傳填好的Excel文件
2. 點擊"生成準考證"
3. 預覽所有準考證
4. 點擊"批量下載PDF"
5. 獲得完整PDF文件
```

### 詳細流程

#### **使用Excel批量導入**
1. ✅ 切換到"Excel批量導入"模式
2. ✅ 點擊"下載Excel模板"獲取模板
3. ✅ 在Excel中填寫40位考生資料
4. ✅ 保存Excel文件
5. ✅ 拖放到上傳區域或點擊選擇
6. ✅ 系統自動解析並顯示考生列表
7. ✅ 檢查信息是否正確
8. ✅ 點擊"生成準考證"
9. ✅ 查看所有準考證預覽
10. ✅ 點擊"批量下載PDF"

#### **使用手動輸入**
1. ✅ 切換到"手動輸入"模式
2. ✅ 填寫考生信息表單
3. ✅ 點擊"➕ 新增考生"
4. ✅ 重複添加其他考生
5. ✅ 檢查考生列表
6. ✅ 點擊"生成準考證"
7. ✅ 批量下載PDF
