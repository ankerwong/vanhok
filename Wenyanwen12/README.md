# 萬鶴書院 - HKDSE 中文科文言文復習平台

## 📚 項目簡介

這是一個專為 HKDSE 中文科考生設計的文言文復習網站，涵蓋考試必讀的 12 篇指定文言經典篇章。網站採用萬鶴書院品牌配色（紫色與黃色），提供完整的原文、字詞釋義、寫作手法分析、DSE 歷年考點等學習資源。

4. **品牌設計**

## 🔍 功能詳解

### 1. 首頁功能
- **考頻標識**：用星級（★）標示歷年考試出現頻率
- **年份標註**：清晰顯示每篇文章在哪些年份出現
- **標籤系統**：展示每篇文章的核心考點（如「比喻論證」「對比手法」等）
- **一頁顯示**：所有12篇文章在一頁內完整展示，無需翻頁

### 2. 文章頁面功能
- **返回導航**：固定在頂部的返回按鈕
- **原文展示**：完整文言文原文，使用優雅的襯線字體
- **字詞釋義**：表格形式，包含字詞、釋義、例句三欄
- **生僻字注音**：提供粵語拼音和普通話拼音
- **寫作手法**：分類解析修辭手法、表現手法、結構特點
- **詳細解讀**：逐段分析，主旨歸納，思想感情
- **精讀筆記**：核心概念、重點難點、名句賞析、考試要點
- **DSE考點**：歷年考試年份、常考題型、真題示例、答題要點

### 3. DSE考點分析
每篇文章包含：
- **歷年考試年份**：清楚標示該篇文章在哪些年份的DSE考試中出現
- **常考題型**：
  - 字詞釋義題
  - 句式分析題
  - 內容理解題
  - 寫作手法題
  - 主旨思想題
- **真題示例**：精選歷年真實考題
- **答題要點**：專業答題指引，幫助學生掌握答題技巧
