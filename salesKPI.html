万鹤书院招生销售岗位绩效管理制度（终稿）

版本：内部执行版
【保密提示】本制度及页面内全部内容仅供万鹤书院内部管理使用，未经授权不得对外传播、转发、截图外流或用于非本机构业务目的。

一、适用对象与保密要求

本制度适用于万鹤书院所有招生销售岗位员工。所有员工应严格遵守本制度，所有内部线索、客户信息、价格政策、系统记录、绩效数据均属于公司内部机密。

二、薪酬结构

销售岗位工资构成为：底薪 + 职级工资 + 绩效工资 + 补贴 + 提成/奖金（如适用）。

  • 底薪、职级工资、补贴：按岗位及职级固定发放；
  • 绩效工资：依据本制度按月考核、次月发放；
  • 超额业绩提成或专项奖金：根据公司另行公布的提成政策执行；
  • 保底业绩对应部分不计额外佣金。

三、保底销售额管理

1. 月度保底销售额

每位销售人员每月必须完成保底销售额，计算公式如下：

月保底销售额 = 固定薪酬部分 × 16.67

其中，固定薪酬部分是指：底薪 + 职级工资 + 绩效工资 + 补贴。

2. 年度保底销售额

年度保底销售额 = 月保底销售额 × 12。

3. 前置超额抵扣

前置月份超额完成保底业绩的，超出部分可在同一自然年度内抵扣后续月份未完成的保底业绩缺口，但仅用于认定保底是否达成，不重复计算提成或奖金。

4. 提前完成年度保底

提前完成年度保底业绩的销售人员，可申请职级评估、晋升储备或更高提成档位评估。

5. 未达标处理

  • 单月未完成保底业绩：进入再培训流程并给予书面预警；
  • 连续两个月未完成保底业绩：纳入重点辅导名单；
  • 连续三个月未完成保底业绩：公司可依据制度及劳动法规进行调岗、降级或解除劳动关系处理。

四、绩效工资评分与折算办法

销售岗位过程绩效统一采用“先评分、后折算”的核算方式。

1. 绩效工资基数

绩效工资基数为员工个人薪酬结构中约定的绩效工资金额，由公司根据岗位和职级标准确定。

2. 过程绩效总分

月度过程绩效满分为100分。

3. 正向考核项目及权重

项目目标值权重分值
朋友圈有效完成天数25天/月11.4分
小红书/视频号有效完成天数25天/月11.4分
直播有效参与次数6次/月27.3分
自产有效线索5条/月22.7分
公司分配线索承接30条/月27.2分

4. 单项得分公式

单项得分 = 该项权重分值 × Min（实际完成值 ÷ 目标值，1）

当实际完成值达到或超过目标值时，该项按满分计算；超过目标值的部分不在当月重复加分，但可作为评优、晋升、年度评估及专项激励参考。

5. 月度绩效工资初算额

月度绩效工资初算额 = 个人绩效工资基数 ×（月度过程绩效总分 ÷ 100）

6. 月度实发绩效工资

月度实发绩效工资 = 月度绩效工资初算额 − 一般SOP月度处罚 − 采购线索专项扣款 − 其他经批准专项扣款。如计算结果低于0元，则按0元计发。

五、各考核指标定义

1. 朋友圈有效完成天数

朋友圈发布按天统计。销售人员在自然日内同时满足以下条件的，当日计为1个有效完成天数：

  1. 当日发布不少于2条朋友圈；
  2. 内容与公司业务相关，包括但不限于课程介绍、活动宣传、学员成果、家长反馈、教学现场、国际教育知识分享、直播预告或招生信息；
  3. 内容完整，至少应为图片/视频配文案，或信息完整、表达清晰的纯文字内容；
  4. 发布后可核查留痕，能够接受公司抽查。

以下情形不计入有效完成天数：当日发布少于2条；内容与业务无关；纯表情、无意义转发、明显凑数内容；短期内大面积重复同一内容；发布后删除导致无法核查；抄袭、盗用素材、虚假宣传或经审核认定不具备传播价值的内容。

2. 小红书/视频号有效完成天数

小红书/视频号发布按天统计。销售人员在自然日内同时满足以下条件的，当日计为1个有效完成天数：

  1. 当日在小红书或视频号任一平台完成不少于1条有效发布；
  2. 内容与公司业务相关，包括但不限于课程介绍、学员案例、家长咨询答疑、国际教育知识科普、活动宣传、直播切片或品牌传播内容；
  3. 内容形式完整：小红书应包含标题、正文、图片/视频；视频号应包含完整视频内容或直播回放内容，并附基本说明文字；
  4. 发布后能够提供链接、截图或后台记录，供公司审核。

以下情形不计入有效完成天数：仅保存草稿、仅测试发布、未实际公开；仅有极短且无信息量内容；文案、标题、图片或视频明显敷衍；短期内大量重复同一内容；发布后删除或隐藏导致无法核查；抄袭、虚假宣传、夸大承诺、违规引流等内容。

3. 直播有效参与

直播主讲、联合出镜、答疑、引导留资及直播后线索承接等构成有效参与。月度目标为6次。

4. 自产线索定义

自产线索是指非公司统一采购分配，而由销售人员通过私域运营、学生/家长转介绍、直播间咨询、新媒体账号咨询及其他自主拓展方式获得，并经系统确认为真实、有效、可继续跟进的新增线索。月度目标为5条。

5. 公司分配线索承接

公司分配线索采取抢单或指派模式，销售须保持在线并及时承接。成功承接并完成规定时限内首次动作的，方可计入当月承接数量。月度目标为30条；若总部当月客观供给不足，可由部门主管下调目标。

6. 内容发布统一审核口径

内容发布类考核以“完成发布动作 + 内容质量合格 + 可核查留痕”三项同时满足为计分前提。涉及学生案例、家长反馈、成绩展示、录取结果等内容时，应符合公司合规要求，不得侵犯客户隐私或发布未经授权内容。

六、小麦助教系统跟进管理要求

1. 跟进人责任

经理录入新线索并在小麦助教系统中指定销售为“跟进人”后，该销售即对该线索承担首次联系、过程记录、持续推进及结果反馈责任。

2. 首次跟进认定

首次跟进是指销售通过电话、微信群、微信、企业微信或其他公司认可方式，对客户实施首次联系动作。已拨打电话、已发送消息、已在群内与客户进行针对性沟通，均视为已实施首次跟进。首次跟进不以客户是否接通、是否回复、是否深入沟通为前提。

3. 跟进记录义务

销售每次实施跟进后，均应在小麦助教系统中及时、真实、完整填写跟进记录。即使客户未接电话、未回复消息或未进行深入沟通，也必须如实留痕。

4. 下次跟进时间

只要线索尚未邀约或尚未关单，销售在每次填写跟进记录时，必须明确填写“下次跟进”的具体时间。该时间必须明确、具体、可执行，不得使用“近期跟进”“后续联系”等模糊表述。

5. 工作失误认定

超过已设定“下次跟进时间”12小时后，如系统内仍未出现新的跟进记录，且无有效报备或豁免事由的，可认定为销售工作失误。

6. 线索脱管认定原则

未跟进的线索、未留痕的跟进、未按约继续推进的客户，均视为脱管线索。公司不以客户是否最终流失作为唯一判断标准，而以销售是否履行持续跟进和系统记录义务作为主要认定依据。

七、一般SOP月度处罚机制

一般SOP失误按月统计并执行处罚。

1. 纳入统计的失误情形

  1. 线索分配后未按要求进行首次跟进；
  2. 已进行首次或后续跟进，但未在小麦助教系统中填写记录；
  3. 跟进记录中未填写明确的下次跟进时间；
  4. 在线索未邀约、未关单的情况下，超过本人上次填写的下次跟进时间12小时后，系统中仍无新的跟进记录；
  5. 其他导致线索推进中断、留痕中断、责任不清的管理失误。

2. 处罚标准

销售人员在自然月内，上述一般SOP失误累计达到3次及以上的，公司给予：

  • 书面警告一次；
  • 当月绩效工资扣减500元。

八、公司采购线索风险管控与专项扣款

公司采购线索按300元/条标准执行专项扣款管理。

1. 适用范围

适用于公司从第三方招生平台、广告平台、合作渠道或其他外部供应商付费采购并分配给销售人员跟进的全部线索。

2. 专项扣款认定情形

  1. 线索分配后未按要求完成首次跟进；
  2. 已完成首次或后续跟进，但未在系统中如实填写记录；
  3. 在线索尚未邀约或关单的情况下，未填写明确下次跟进时间；
  4. 超过约定下次跟进时间12小时后，系统内仍无新的跟进记录；
  5. 其他导致采购线索脱管、失跟、后台无留痕的情形。

只要销售未按要求持续跟进并在后台形成完整留痕，即适用本条款。

3. 专项扣款标准

经认定属于前述情形的，每条采购线索扣减300元绩效工资。

4. 与一般SOP处罚的衔接

对同一条采购线索、同一事实，实行“就高不重复”原则。已按采购线索专项条款扣减300元的，不再将该同一事实重复计入一般SOP月度处罚次数。

5. 豁免情形

如因系统故障、平台异常、公司统一安排导致短时无法录入，且销售已第一时间向主管报备并在恢复后24小时内补录，可申请豁免。异常线索本身存在明显错误、重复、虚假、号码无效等情形的，须在规定时间内完成申诉留痕。

九、记录与反馈要求

1. 每日记录

销售须于每日下班前提交当日工作记录，包括新增线索数、有效跟进数、邀约数、成交数/金额、重点客户推进情况等。

2. 每周反馈

销售须每周提交一次复盘，至少包括线索来源分析、转化率分析、丢单原因、下周重点推进名单及需主管支持事项。

3. 月度复盘

销售须于月末提交月度总结，至少包括业绩完成情况、过程指标完成情况、问题总结及改进计划。虚报、瞒报、伪造数据，按红线行为处理。

十、存量线索激活

每月或每次推出新课程、新活动、新产品时，销售须对其负责的存量家长群及客户池进行定向触达，包括但不限于群发课程信息、活动海报、试听邀约及重点客户一对一激活。超时未发、漏发或虚假发送的，按公司管理要求处理；截图造假、故意漏发导致业务损失的，按严重违规或红线行为处理。

十一、红线行为

以下行为属于红线行为。第一次发现，当月绩效工资扣减50%，并给予严重警告；第二次发现，公司有权依据制度和劳动法规解除劳动关系。涉及违法、侵占公司财产、损害公司重大利益或造成恶劣影响的，公司有权直接从重处理，并保留追究责任的权利。

1. 客户与成交类

  1. 私单、飞单、绕开公司成交；
  2. 私自收款、代收款、截留学费，或诱导客户向个人转账；
  3. 伪造客户签约、伪造成交、伪造付款凭证、拆单并单骗取绩效；
  4. 将本应进入公司体系的客户、订单或课程需求转移至个人或第三方成交。

2. 线索与系统数据类

  1. 冒领线索、盗用客户、隐匿客户不回流系统；
  2. 删除、篡改、伪造系统跟进记录或授意他人篡改；
  3. 泄露客户信息、价格体系、内部资料、招生策略；
  4. 故意不录入系统、故意隐瞒客户状态、规避公司监管。

3. 师资与课程资源类

  1. 私自将外部学生、家长或潜在客户介绍给公司老师接私单；
  2. 私下撮合公司老师与家长直接交易，绕开公司正常招生、签约、收费、排课或服务流程；
  3. 擅自向客户透露公司老师个人联系方式，用于脱离公司体系建立交易关系；
  4. 组织、协助、默认或隐瞒公司老师私下接单、私下授课、私下收费等行为；
  5. 从老师私单、外部授课、非公司体系内课程交易中收取介绍费、回扣、分成或其他利益；
  6. 借公司品牌、老师资历、课程方案、教学资源对外承接个人业务或第三方业务；
  7. 将公司在读学员、家长、课程需求导流至非公司正式业务体系成交；
  8. 销售人员与老师、教务、合作方或其他内部员工串联，共同实施私单承接、客户转移、排课外流、私下收费等行为。

4. 品牌与职业操守类

  1. 未经审批擅自承诺折扣、退款、赠课、保录、保结果等；
  2. 对客户实施辱骂、骚扰、威胁、虚假宣传或严重误导；
  3. 恶意抢单、压单、藏单，或为竞争机构导流；
  4. 兼职竞争机构、为竞争机构提供客户、线索、师资或业务支持；
  5. 其他严重损害公司品牌、业务秩序或经营利益的行为。

上述行为的认定，不以最终是否实际成交、是否实际收款、是否已正式开课为前提。只要存在私下介绍、撮合、导流、隐瞒、协助脱离公司体系交易的事实，即可认定构成红线行为。

十二、绩效发放、解释与附则

1. 发放原则

绩效工资按月考核、次月发放；所有数据以小麦助教系统、直播记录、内容发布留痕、截图佐证、主管审核结果及公司相关后台数据为准。

2. 数据争议

对绩效数据有争议的，以公司系统记录、主管复核结果及经审批的异常报备为最终依据。

3. 制度解释权

本制度由万鹤书院管理层与人力资源部门共同解释，并可根据业务发展、组织管理需要进行修订更新。
