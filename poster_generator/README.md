# 万鹤书院 - 教师海报生成器

一个专业的教师宣传海报在线生成工具，专为万鹤书院设计。销售人员可以快速创建高质量的教师介绍海报，无需任何设计经验。

## 🎯 项目简介

万鹤书院是一所专门从事香港及国际课程教育的机构。本工具旨在帮助销售团队快速生成专业、统一品牌形象的教师宣传海报，用于各类营销渠道。

### 品牌色彩
- **主题色**: 紫色 (#3c2479) - 代表专业与学术
- **点缀色**: 黄色 (#f0c95a) - 象征活力与创新

## ✨ 核心功能

### 1. 信息输入
- ✅ **教师姓名** - 支持中英文输入
- ✅ **职称/头衔** - 可选项，如"资深数学导师"
- ✅ **教师照片** - 支持JPG/PNG格式，建议使用透明底照片
- ✅ **教师简介** - 支持多行point form格式，自动计数（限500字）

### 2. 布局选择
- 🖼️ **横版海报** (16:9, 1920×1080px)
  - 适合：微信公众号横图、网站Banner、电脑端展示
  - 布局：左侧照片，右侧内容
  
- 📱 **竖版海报** (9:16, 1080×1920px)
  - 适合：朋友圈分享、Instagram Story、手机端传播
  - 布局：上方照片，下方内容

### 3. 风格模板

#### 现代风格
- 紫色渐变背景
- 大胆的视觉设计
- 黄色装饰圆形元素
- 适合年轻、活力的教师形象

#### 经典风格
- 白色背景搭配紫色区块
- 圆形照片设计
- 对称式排版
- 适合资深、专业的教师形象

### 4. 实时预览
- 所有修改即时显示在预览区
- 高清Canvas渲染
- 所见即所得

### 5. 一键导出
- 高清PNG格式（原始分辨率）
- 自动文件命名：`教师姓名_海报_横版/竖版.png`
- 适合直接用于打印和线上传播

## 🚀 使用指南

### 基本流程
1. **填写教师信息**
   - 输入教师姓名（必填）
   - 可选填写职称
   - 上传教师照片（必填，建议透明底PNG）
   - 填写教师简介（必填，每行一个要点）

2. **选择海报样式**
   - 选择横版或竖版布局
   - 选择现代或经典风格

3. **预览并调整**
   - 在右侧预览区实时查看效果
   - 根据需要调整内容

4. **下载海报**
   - 点击"下载海报"按钮
   - 保存高清PNG文件

### 输入建议

#### 教师简介格式
建议使用point form格式，每行一个要点：
```
• 香港中文大学教育学硕士
• 10年以上DSE数学教学经验
• 帮助超过200名学生考获5**成绩
• 专注IGCSE/A-Level数学课程
```

可以使用以下符号开头（会自动识别并替换为装饰符号）：
- `•` 圆点
- `-` 短横线
- `*` 星号

#### 照片要求
- **最佳格式**: 透明底PNG
- **替代方案**: 纯色背景JPG（建议白色或浅色背景）
- **分辨率**: 建议至少 800×800px
- **比例**: 正方形或人像比例最佳
- **内容**: 清晰的正面或半侧面照，专业着装

## 🛠️ 技术实现

### 技术栈
- **HTML5** - 语义化结构
- **CSS3** - 现代样式与响应式设计
- **JavaScript (原生)** - 核心逻辑实现
- **Canvas API** - 高清海报渲染

### 核心功能实现
- **图片处理**: FileReader API + Canvas drawImage
- **透明图处理**: 完整保留PNG透明通道
- **文字渲染**: Canvas 2D Context文字API
- **图片缩放**: 自适应缩放算法，保持比例
- **高清导出**: Canvas.toBlob生成高质量PNG

### 常见问题

**Q: 上传照片后没有显示？**
A: 请确保照片格式为JPG或PNG，且文件大小不超过10MB。

**Q: 导出的海报模糊？**
A: 本工具导出的是全高清分辨率(1920×1080或1080×1920)，如果显示模糊可能是查看软件的问题，建议用专业图片查看器打开。

**Q: 可以修改品牌色吗？**
A: 当前版本固定使用万鹤书院品牌色，确保品牌一致性。未来版本可能开放自定义功能。

**Q: 简介文字太多无法完全显示？**
A: 建议简介控制在4-6个要点，每个要点简明扼要。文字过多会影响视觉效果。
