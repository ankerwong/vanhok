# 快速使用指南

## 🚀 5分钟快速上手

### 第一步：打开网页
在浏览器中打开 `index.html` 文件，您将看到万鹤书院教师海报生成器的界面。

### 第二步：填写教师信息

#### 必填信息 ✅
1. **教师姓名**: 输入教师的中文或英文名字
   - 示例: "张老师" 或 "Mr. Zhang"

2. **教师照片**: 点击上传区域选择照片
   - 💡 **最佳效果**: 使用透明底PNG图片
   - 📸 **替代方案**: 白色或浅色背景的JPG照片
   - 📏 **推荐尺寸**: 至少 800×800 像素

3. **教师简介**: 输入教师的资历和特点（每行一个要点）
   ```
   • 香港中文大学教育学硕士
   • 10年以上DSE数学教学经验
   • 帮助超过200名学生考获5**成绩
   • 专注IGCSE/A-Level数学课程
   ```

#### 可选信息 ⭐
- **职称/头衔**: 如 "资深数学导师"、"首席英语讲师" 等

### 第三步：选择海报样式

#### 海报方向
- **横版** (16:9): 适合网站横幅、PPT、微信公众号横图
- **竖版** (9:16): 适合朋友圈分享、Instagram Story、手机展示

#### 海报风格
- **现代风格**: 紫色渐变背景，视觉冲击力强，适合年轻活力的形象
- **经典风格**: 白色背景配紫色区块，专业典雅，适合资深教师

### 第四步：实时预览
- 在右侧预览区域可以实时看到海报效果
- 修改任何信息都会立即更新预览
- 可以随时切换横版/竖版或现代/经典风格

### 第五步：下载海报
1. 确认预览效果满意
2. 点击 **"下载海报"** 按钮
3. 海报将以PNG格式自动下载，文件名格式为：`教师姓名_海报_横版.png`

## 💡 实用技巧

### 照片处理建议

#### 如何制作透明底照片？
如果您没有透明底照片，可以使用以下工具：
- **在线工具**: remove.bg (https://remove.bg)
- **手机APP**: 醒图、美图秀秀等
- **电脑软件**: Photoshop、GIMP等

#### 照片拍摄要求
- ✅ 正面或半侧面
- ✅ 专业着装
- ✅ 光线充足
- ✅ 背景简洁
- ❌ 避免杂乱背景
- ❌ 避免阴影过重

### 简介撰写技巧

#### 优秀示例 ✨
```
• 香港大学教育学博士
• 15年DSE英语教学经验
• 学生DSE英语平均分5+
• 出版3本DSE英语备考书籍
• 前香港名校英语科主任
```

#### 要点提示
- 📝 3-6个要点最佳
- 📝 每个要点一行，简明扼要
- 📝 突出学历、经验、成绩
- 📝 使用具体数据更有说服力
- 📝 可以使用 • - * 开头（会自动格式化）

## 🔧 常见问题解决

### 问题1: 上传的照片显示不出来
**解决方案**:
- 检查照片格式是否为JPG或PNG
- 确认文件大小不超过10MB
- 尝试使用其他照片

### 问题2: 简介文字显示不全
**解决方案**:
- 减少简介要点数量（建议4-6条）
- 简化每条要点的文字
- 删除不必要的修饰词

### 问题3: 下载的海报打开是模糊的
**解决方案**:
- 海报是高清的（1920×1080或1080×1920）
- 可能是查看软件的问题
- 建议用专业图片查看器或直接上传使用

### 问题4: 照片在海报中被裁切了
**解决方案**:
- 这是正常的，系统会自动缩放以填充区域
- 建议使用正方形或人像比例的照片
- 照片主体应该居中

### 问题5: 可以批量生成多个教师的海报吗？
**当前版本说明**:
- 暂不支持批量生成
- 每次需要单独填写并下载
- 但操作很快，一个海报只需1-2分钟
