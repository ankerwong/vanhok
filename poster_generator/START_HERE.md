# 🎓 万鹤书院教师海报生成器

欢迎使用万鹤书院教师海报生成器！

## 🚀 快速开始

### 方法1: 直接使用（推荐）
1. 双击打开 `index.html` 文件
2. 或在浏览器中打开该文件
3. 开始使用！

### 方法2: 本地服务器（可选）
如果您想使用本地服务器运行：
```bash
# Python 3
python -m http.server 8000

# 然后访问 http://localhost:8000
```

## 🎯 主要功能

✨ **输入教师信息** - 姓名、职称、照片、简介  
✨ **选择布局** - 横版(16:9) 或 竖版(9:16)  
✨ **选择风格** - 现代风格 或 经典风格  
✨ **实时预览** - 所有修改立即可见  
✨ **一键下载** - 高清PNG格式，直接使用

## 💡 快速提示

### 照片要求
- 建议使用**透明底PNG**
- 尺寸至少 800×800 像素
- 正面或半侧面，专业着装

### 简介格式
每行一个要点，使用 • - * 开头：
```
• 香港中文大学教育学硕士
• 10年以上DSE数学教学经验
• 帮助超过200名学生考获5**成绩
```
