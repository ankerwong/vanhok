## 🚀 快速開始（3步驟）

### 步驟1：準備Excel數據

**下載模板**：打開系統 → 點擊「下載Excel模板」

**填寫要點**：
```
同一考生多行，每行一個科目，考生編號保持一致

示例：
姓名      | 考生編號  | 科目              | 考試日期
張偉明    | A123456  | English Language  | 2025-04-15
張偉明    | A123456  | Chinese Language  | 2025-04-16
張偉明    | A123456  | Mathematics       | 2025-04-17
```

**重要**：
- ✅ 日期格式：`2025-04-15`（YYYY-MM-DD）
- ✅ 相同考生使用相同考生編號
- ✅ 每行填寫一個科目

### 步驟2：上傳並生成

```
1. 上傳Excel文件
2. 系統自動合併同一考生的多科目
3. 顯示：「10位考生（28科）」
4. 點擊「生成準考證」
```

### 步驟3：選擇性下載（NEW！）

**情況A：少量考生（1-10人）**
```
→ 直接點擊「下載全部」✅
```

**情況B：大量考生（10+人）**⭐ 推薦
```
1. 勾選前10位考生
2. 點擊「下載選中 (10)」
3. 取消全選
4. 勾選接下來10位
5. 再次下載
6. 重複直到完成 ✅
```

## 📋 使用技巧

### 技巧1：Excel準備

**推薦格式**：
```excel
姓名 | 英文姓名 | 考生編號 | 性別 | 科目 | 應考語言 | 考場 | 考試日期 | 考試時間
```

**常見錯誤**：
- ❌ 同一考生使用不同編號 → 會生成多張準考證
- ❌ 日期格式不標準 → 可能顯示錯誤
- ❌ 有空白行 → 導入時會跳過

**最佳實踐**：
- ✅ 使用提供的模板
- ✅ 複製粘貼基本信息
- ✅ 只修改科目相關字段

### 技巧2：分批下載策略

**小批量（1-10人）**：
```
→ 直接「下載全部」
→ 時間：約5秒
→ 文件：10-12 MB
```

**中批量（11-20人）**：
```
→ 分2批下載（每批10人）
→ 時間：約10秒（總計）
→ 文件：2個PDF，各10-12 MB
```

**大批量（21-40人）**：
```
→ 分4批下載（每批10人）
→ 時間：約20秒（總計）
→ 文件：4個PDF，各10-12 MB
```

**超大批量（40+人）**：
```
→ 分批下載，每批5-10人
→ 根據電腦性能調整
→ 如遇到問題，減少每批數量
```

### 技巧3：文件管理

**自動命名格式**：
- 選中下載：`HKDSE準考證_選中_10人_2026-02-06.pdf`
- 全部下載：`HKDSE準考證_全部_40人_2026-02-06.pdf`

**建議命名**：
```
批次1：HKDSE準考證_A班_1-10號.pdf
批次2：HKDSE準考證_A班_11-20號.pdf
批次3：HKDSE準考證_B班_1-10號.pdf
```

**合併PDF（可選）**：
- 工具1：Adobe Acrobat
- 工具2：在線PDF合併（如 smallpdf.com）
- 工具3：命令行（pdftk）

### 技巧4：打印設置

**推薦設置**：
```
紙張：A4
方向：縱向
縮放：100%（實際大小）
邊距：默認（12mm）
質量：標準（無需高質量）
雙面：否
```

**打印測試**：
- ✅ 先打印1張測試
- ✅ 檢查條形碼可掃描
- ✅ 檢查文字清晰
- ✅ 確認無超出邊界

## ⚠️ 常見問題

### Q1：下載失敗怎麼辦？
**A**：使用選擇性下載，減少每批數量
```
建議：
- 首次：選5人測試
- 正常：每批10人
- 問題：減至每批5人
```

### Q2：日期顯示01-JAN？
**A**：Excel日期格式問題，重新設置：
```
1. 選中日期列
2. 設置格式為「文本」
3. 輸入：2025-04-15
4. 重新上傳
```

### Q3：一個考生生成多張準考證？
**A**：考生編號不一致
```
檢查：
- A123456 ✅
- A123456 ✅
- A123457 ❌（不同編號）
```

### Q4：條形碼掃不出來？
**A**：v3.0已修復，如仍有問題：
```
1. 檢查打印質量
2. 確保紙張平整
3. 清潔掃描器
4. 嘗試不同掃描角度
```

### Q5：PDF文件過大？
**A**：v3.0已優化至每人1.2MB
```
如仍過大：
- 檢查是否使用v3.0版本
- 查看文件屬性確認大小
- 嘗試重新生成
```

### Q6：如何合併多個PDF？
**A**：使用外部工具
```
在線工具（推薦）：
- https://smallpdf.com/merge-pdf
- https://www.ilovepdf.com/merge_pdf

桌面軟件：
- Adobe Acrobat DC
- PDFtk（命令行）
```
