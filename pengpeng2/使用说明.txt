=====================================================
     面试提词器 2.0 - 快速使用说明
=====================================================

🎯 3 步开始使用

第一步：打开应用
  → 双击 index.html 文件

第二步：输入内容
  → 粘贴你的面试问答到文本框
  → 或点击"加载示例"查看格式

第三步：开始练习
  → 点击"开始使用"按钮
  → 自动进入提词器界面

=====================================================

📝 内容格式要求

必须使用序号标记问题：

1、第一个问题的标题
这里是回答内容，可以很长...
支持中英文混合。

2、第二个问题的标题  
继续回答内容...

3、第三个问题的标题
更多内容...

✅ 支持的序号格式：1、 1. 1) 1）1]

=====================================================

⌨️ 常用快捷键

空格 = 播放/暂停
↑↓ = 上一句/下一句
R = 重置到开头
F = 全屏模式
E = 编辑内容
1-9 = 跳转到对应问题

=====================================================

🎨 主要功能

✅ Fading 聚焦 - 当前句子高亮，其他句子淡化
✅ 三速调节 - 慢/中/快 (5.5s / 3.5s / 2.5s)
✅ 字号调节 - 正常/大/超大
✅ 主题切换 - 亮色/暗色
✅ 自动保存 - 下次打开自动恢复内容
✅ 问题导航 - 点击左侧列表快速跳转
✅ 进度显示 - 实时显示完成百分比

=====================================================

💡 使用技巧

1. 第一次练习 → 使用"慢速"模式
2. 熟悉后 → 使用"中速"模式
3. 快速复习 → 使用"快速"模式
4. 专注阅读 → 按 F 进入全屏
5. 针对练习 → 点击左侧问题跳转

🆘 遇到问题？

问题1：点击"开始使用"无反应
→ 检查是否输入了带序号的问题

问题2：内容没有保存
→ 检查浏览器是否允许 localStorage

问题3：效果不明显
→ 尝试切换主题或调整字号

=====================================================

最佳分辨率：1920×1080
