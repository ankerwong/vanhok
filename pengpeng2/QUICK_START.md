# 🚀 快速上手指南

## 5 分钟开始使用面试提词器 2.0

---

## 第一步：打开应用 (30 秒)

### 方法 1：双击打开
1. 找到 `index.html` 文件
2. 双击打开
3. 浏览器会自动加载

### 方法 2：拖放打开
1. 打开任意浏览器
2. 将 `index.html` 拖入浏览器窗口
3. 自动加载应用

**推荐浏览器：** Chrome / Edge

---

## 第二步：准备内容 (2 分钟)

### 选项 A：使用示例内容

**最快体验方式：**
1. 点击页面上的 **"加载示例"** 按钮
2. 自动填充示例内容
3. 直接进入下一步

### 选项 B：输入自己的内容

**准备你的面试问答：**

```text
1、自我介绍
您好，我叫张三，毕业于某某大学...

Hello, my name is Zhang San...

2、为什么选择这个职位？
我对这个职位很感兴趣，因为...

I am interested in this position because...

3、你的优势是什么？
我的主要优势包括...

My main strengths include...
```

**格式要点：**
- ✅ 使用序号：`1、` `2、` `3、`
- ✅ 序号后写问题标题
- ✅ 下面写回答内容
- ✅ 支持中英文混合
- ✅ 问题之间空行分隔

**操作步骤：**
1. 复制你的面试内容
2. 粘贴到大文本框
3. 看到问题数量更新（如"5 个问题"）

---

## 第三步：开始使用 (30 秒)

### 确认并启动

1. 检查右下角显示：
   ```
   XXX 字符    X 个问题
   ```

2. 点击蓝色的 **"开始使用"** 按钮

3. 系统会：
   - ✅ 自动解析你的内容
   - ✅ 分离问题和句子
   - ✅ 进入提词器界面
   - ✅ 自动开始播放

---

## 第四步：基本操作 (2 分钟)

### 界面布局快速认识

```
┌────────────────────────────────────┐
│ 🎤 [▶️] [慢|中|快] [Aa] [🌐] [🌙] ...│ ← 顶部控制栏
├────┬───────────────────────────────┤
│1.问题│                             │
│2.问题│      Fading 内容区          │ ← 主显示区
│3.问题│      ▓ 当前句子 ▓           │
│...  │                             │
├────┴───────────────────────────────┤
│ 问题标题  ▓▓▓▓░░░░  65%  02:15/03:30│ ← 底部进度栏
└────────────────────────────────────┘
```

### 核心操作

#### 1. 播放控制
- **播放/暂停**：点击 ▶️ 按钮（或按空格键）
- **自动滚动**：句子会自动切换并高亮

#### 2. 速度调节
- **慢速** (5.5秒/句)：初学阶段
- **中速** (3.5秒/句)：日常练习 ⭐ 推荐
- **快速** (2.5秒/句)：快速复习

点击对应按钮即可切换

#### 3. 快速跳转
- **点击左侧问题列表**：立即跳转到该问题
- **按数字键 1-9**：跳转到对应序号的问题

#### 4. 字号调节
- 点击 **Aa** 按钮循环切换：
  - 正常（默认）
  - 大
  - 超大

## 🎯 实战演示

### 场景 1：第一次练习

**目标：** 熟悉全部内容

1. 选择 **慢速** 模式
2. 点击 **▶️ 播放**
3. 专注阅读高亮的当前句子
4. 跟随自动滚动
5. 从头到尾完整阅读

**预计时间：** 根据内容量，约 10-20 分钟

---

### 场景 2：针对性练习

**目标：** 加强某个问题

1. 在左侧列表点击要练习的问题
2. 立即跳转到该问题
3. 选择 **中速** 模式
4. 反复阅读该问题的回答
5. 按 `↓` 键逐句推进，按需暂停思考

**预计时间：** 每个问题 2-3 分钟

---

### 场景 3：快速复习

**目标：** 考前快速浏览

1. 选择 **快速** 模式
2. 点击 **▶️ 播放**
3. 快速浏览所有内容
4. 使用数字键 1-9 快速跳转
5. 重点问题点击左侧列表定位

**预计时间：** 5-8 分钟完整复习

---

### 场景 4：模拟面试

**目标：** 脱稿练习表达

1. 选择 **中速** 或 **快速**
2. 播放到一个问题后 **暂停**（按空格）
3. 尝试用自己的话复述
4. 继续播放对照答案
5. 重复练习不熟悉的部分

**预计时间：** 每个问题 5 分钟

## 💡 3 个高效技巧

### 技巧 1：善用暂停

**不要一直播放：**
- ⏸️ 遇到难点立即暂停
- 💭 思考回答逻辑
- 🗣️ 尝试口述表达
- ✅ 确认理解后继续

### 技巧 2：利用 Fading 效果

**Fading 是核心优势：**
```
[淡] 上上句
[中] 上一句
[亮] ★ 当前句子 ★  ← 只看这里！
[中] 下一句
[淡] 下下句
```

**专注技巧：**
- 👁️ 只看最亮的句子
- 📖 用余光感知前后句
- 🚫 完全忽略很淡的句子

### 技巧 3：分阶段调速

**循序渐进：**
```
第 1-2 天 → 慢速 → 理解内容
第 3-5 天 → 中速 → 强化记忆  
第 6-7 天 → 快速 → 快速复习
```

## 🆘 遇到问题？

### 问题 1：点击"开始使用"无反应

**可能原因：**
- 没有检测到问题

**解决方法：**
1. 检查内容格式（必须有序号）
2. 确保至少有 1 个问题
3. 查看是否显示"X 个问题"

### 问题 2：Fading 效果不明显

**可能原因：**
- 浏览器不支持或主题问题

**解决方法：**
1. 使用 Chrome 或 Edge 浏览器
2. 尝试切换亮色/暗色主题
3. 调整字号到"大"或"超大"

### 问题 3：滚动太快或太慢

**解决方法：**
1. 点击顶部的速度按钮
2. 或使用 `↑/↓` 键手动控制
3. 暂停后用鼠标点击句子跳转

### 问题 4：内容没有保存

**解决方法：**
- 内容会自动保存到本地
- 下次打开自动恢复
- 如果未恢复，检查是否在同一浏览器
