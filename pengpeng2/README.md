## 📖 项目简介

面试提词器 2.0 是一款专为面试准备设计的 Web 应用。它通过智能的 **Fading 聚焦效果**、**自动滚动系统** 和 **动态内容管理**，帮助你在面试时更专注、更自信地表达。

### 🎯 核心优势

- ✅ **零配置启动** - 直接粘贴内容即可使用，无需复杂设置
- ✅ **智能聚焦** - 独特的 Fading 效果让你专注于当前句子
- ✅ **自适应布局** - 专为 1920×1080 优化，一屏展现更多内容
- ✅ **完整控制** - 顶部集中控制栏，一键调节所有参数
- ✅ **数据持久化** - 本地自动保存，下次打开自动恢复
- ✅ **响应式设计** - 支持多种屏幕尺寸

## ✨ 功能特性

### 1️⃣ 动态内容输入系统

**启动页面功能：**
- 📝 粘贴面试内容到文本框
- 🔍 自动识别问题和答案
- 📊 实时显示字符数和问题数量
- 💾 自动保存到本地存储
- 📋 一键加载示例内容

**支持的格式：**
```text

回答内容...

2、第二个问题
回答内容...
```

支持的序号格式：`1、` `1.` `1)` `1）` `1]`

---

### 2️⃣ Fading 聚焦效果

**核心创新功能：**

- **当前句子**：100% 不透明度，加粗，字号最大
- **相邻句子**：70% 不透明度，辅助阅读
- **较远句子**：35% 不透明度，提供上下文
- **很远句子**：15% 不透明度，最小化干扰

**效果演示：**
```
[15%] 很远的句子，几乎不可见
[35%] 较远的句子，轻微可见
[70%] 相邻的句子，清晰可读
[100%] ★ 当前句子 - 高亮加粗，视觉焦点 ★
[70%] 相邻的句子，清晰可读
[35%] 较远的句子，轻微可见
[15%] 很远的句子，几乎不可见
```

**平滑过渡：**
- 使用 CSS 贝塞尔曲线动画
- 过渡时间：0.5 秒
- 自然流畅的视觉体验

## 📘 使用指南

### 第一步：准备内容

**格式要求：**
```text
1、自我介绍
您好，我叫张三。我毕业于XX大学，专业是市场营销。

Hello, my name is Zhang San. I graduated from XX University majoring in Marketing.

2、为什么选择这个职位？
我对这个职位的兴趣源于...

I am interested in this position because...
```

**重要提示：**
- ✅ 使用序号标记问题（1、2、3...）
- ✅ 支持中英文混合
- ✅ 段落间用空行分隔
- ✅ 每个问题可以包含多个段落

---

### 第二步：调整设置

**进入提词器后：**

1. **速度调节**
   - 初次练习：选择"慢速"
   - 熟悉后：选择"中速"
   - 快速复习：选择"快速"

2. **字号调节**
   - 坐得近：选择"正常"
   - 中等距离：选择"大"
   - 坐得远：选择"超大"

3. **主题选择**
   - 白天/明亮环境：亮色主题
   - 夜间/低光环境：暗色主题

---

### 第三步：练习技巧

**有效练习方法：**

#### 阶段一：熟悉内容（第 1-3 天）
- 🐢 使用慢速
- 📖 完整阅读 2-3 遍
- ✏️ 理解每个回答的逻辑
- 🎯 标记重点和难点

#### 阶段二：强化记忆（第 4-7 天）
- 🚶 使用中速
- 🔄 中英文交替练习
- 🗣️ 尝试跟读或复述
- 📹 录制练习视频

#### 阶段三：模拟实战（第 8-10 天）
- 🏃 使用快速
- 🎭 模拟真实面试场景
- 💪 提升表达流畅度
- 😌 建立自信心

---

### 第四步：面试当天

**最后准备：**
- ⏱️ 面试前 30 分钟快速浏览
- 🎯 重点问题再次练习
- 📊 检查关键数据和案例
- 😊 保持放松和自信

### 关键算法

#### 1. 智能分句算法
```javascript
// 特殊处理：
- 小数点识别（3.61）
- 英文缩写识别（Mr. Dr.）
- 中英文混合文本
- 多种句子结束标记
```

#### 2. Fading 效果算法
```javascript
distance = |currentIndex - sentenceIndex|
if distance == 0:  opacity = 1.0, fontSize = 30px
if distance <= 2:  opacity = 0.7, fontSize = 24px
if distance <= 4:  opacity = 0.35, fontSize = 22px
if distance > 4:   opacity = 0.15, fontSize = 20px
```

#### 3. 自动滚动算法
```javascript
setInterval(() => {
    currentIndex++
    updateFading()
    scrollToCenter()
}, speedConfig[speed])
```

## ❓ 常见问题

### Q1: 如何开始使用？
**A:** 直接打开 `index.html`，粘贴你的面试内容，点击"开始使用"即可。

### Q2: 支持哪些内容格式？
**A:** 支持使用序号标记的问答格式（1、2、3... 或 1. 2. 3...），支持中英文混合。

### Q3: 数据会保存吗？
**A:** 会自动保存到浏览器本地存储，下次打开自动恢复。

### Q4: 如何修改内容？
**A:** 点击顶部的"编辑"按钮（或按 E 键）返回输入页面修改。

### Q5: Fading 效果是什么？
**A:** 当前句子高亮显示，其他句子按距离渐隐，帮助你专注于当前内容。

### Q6: 支持移动设备吗？
**A:** 支持，但推荐在电脑上使用以获得最佳体验。

### Q7: 需要网络连接吗？
**A:** 需要联网加载字体图标库（Font Awesome），其他功能可离线使用。

### Q8: 如何导出/分享内容？
**A:** 目前不支持导出，建议保存原始文本到文档。
