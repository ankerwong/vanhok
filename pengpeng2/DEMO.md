#### 演示 1: 播放控制

**操作序列：**

```
1. 自动播放状态 (默认)
   [⏸️] 暂停按钮
   ↓
   当前句子每 3.5 秒自动切换
   ↓
   Fading 效果平滑过渡

2. 点击暂停 (或按空格)
   [▶️] 播放按钮
   ↓
   滚动停止
   ↓
   当前句子保持高亮

3. 点击播放 (或按空格)
   [⏸️] 暂停按钮
   ↓
   恢复自动滚动
```

## 💡 高级技巧演示

### 技巧 1: 组合使用快捷键

**场景：快速定位并精读**

```
操作组合：
1. 按 "5"（跳转到第 5 个问题）
2. 按 空格（暂停）
3. 按 ↓↓↓（向下翻 3 句）
4. 再按 空格（继续播放）
5. 从当前位置开始阅读
```

**场景：循环练习单个问题**

```
操作组合：
1. 点击左侧"3. 优势分析"
2. 完整阅读到该问题结束
3. 再次点击"3. 优势分析"
4. 重复 3-5 次
5. 直到完全熟练
```

---

### 技巧 2: 利用 Fading 提升专注力

**训练方法：**

```
初级阶段：
- 眼睛聚焦在最亮的句子
- 尝试忽略周围淡色句子
- 跟随句子自动切换

进阶阶段：
- 只看当前高亮句子
- 用余光感知前后句
- 形成自然的阅读节奏

高级阶段：
- 当前句子快速识别
- 提前预判下一句内容
- 建立流畅表达习惯
```

---

### 技巧 3: 分段练习策略

**10 个问题的分段方案：**

```
第 1 段 (问题 1-3)：个人介绍类
- 时长：约 5 分钟
- 重点：流畅表达个人经历
- 练习次数：5-8 次

第 2 段 (问题 4-7)：能力展示类
- 时长：约 7 分钟
- 重点：突出核心优势
- 练习次数：8-10 次

第 3 段 (问题 8-10)：互动提问类
- 时长：约 3 分钟
- 重点：展现思考深度
- 练习次数：3-5 次
```
