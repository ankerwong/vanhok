# HKDSE Essay Memorizer

**Vanhok Academy Writing System v1.0**  
**Designed by Anker Wong | Academic Director**

---

## 📖 项目简介

HKDSE Essay Memorizer 是一个专为香港中学文凭考试(HKDSE)英语写作课程设计的互动式范文学习系统。本系统由万鹤书院(Vanhok Academy)学术总监 Anker Wong 设计开发，旨在通过科学的三阶段教学法，帮助学生高效背诵和理解高分范文。

### 🎯 设计理念

研究表明，背诵高分范文是HKDSE英语写作提分最高效的方式之一。本系统将传统的背诵方法系统化、科学化，通过三个递进的学习阶段，帮助学生：
1. **理解文章结构** - 掌握逻辑框架
2. **分析句式语法** - 学习高级表达
3. **强化记忆背诵** - 逐句掌握内容

---

## ✨ 核心功能

### 🔷 阶段一：逻辑框架展示
- **一键显示核心句** - 快速提取每段的主题句(Topic Sentences)
- **段落类型标识** - 清晰标注Introduction、Body Paragraphs、Conclusion
- **渐进式展示** - 核心句逐个淡入，便于教师授课讲解
- **完整原文展示** - 保留完整的范文内容供参考

**教学目标**：帮助学生树立文章的逻辑框架，理解每段的核心观点和论证结构。

### 🔷 阶段二：句式拆解与词汇扩展

提供三种视图模式，可自由切换：

#### 1️⃣ 原文视图
- 展示纯净的原文，适合初次阅读
- 按句子编号组织（Sentence 1-33）

#### 2️⃣ 句式拆解视图
- **主谓宾颜色高亮**：
  - 🔵 **主语 (Subject)** - 蓝色背景
  - 🟢 **谓语 (Verb)** - 绿色背景
  - 🟠 **宾语 (Object)** - 橙色背景
- **语法结构可视化** - 帮助学生理解句子骨架
- **颜色图例说明** - 顶部显示清晰的颜色对照

#### 3️⃣ 词汇扩展视图
- **50+ 重点词汇**高亮显示
- **150+ 同义词替换**，直接显示在词汇下方
- 帮助学生扩展词汇量，学习多样化表达

**教学目标**：深度分析句式结构，掌握高级语法和词汇表达。

### 🔷 阶段三：背诵练习（全新交互方式）

#### 🆕 点击式逐句背诵系统

**交互流程**：
1. 点击"开始背诵"启动练习
2. 点击按钮第一次 → 显示当前句子的**SVO框架**（主谓宾提示）
3. 点击按钮第二次 → 显示该句**完整内容**
4. 重复步骤2-3，逐句完成全文背诵

**核心特性**：
- ✅ **累积显示** - 所有已背诵的完整句子保留在屏幕上
- ✅ **卡片式布局** - 每句一个独立卡片，结构清晰
- ✅ **实时进度跟踪** - 进度条 + 句子计数 (X/33)
- ✅ **智能提示** - SVO框架帮助回忆句子结构
- ✅ **完成庆祝** - 背完全文后显示祝贺消息
- ✅ **快捷键支持** - 按空格键快速切换（可选）
- ✅ **重置功能** - 随时重新开始练习

**教学目标**：通过渐进式提示和累积显示，帮助学生逐句背诵并掌握全文。

## 🚀 使用方法

### 快速开始
1. 双击 `index.html` 文件
2. 浏览器自动打开，系统即可使用
3. 无需安装任何软件或服务器

### 推荐教学流程

#### 第一课时（30分钟）- 理解结构
1. 进入**阶段一**
2. 点击"显示核心句"
3. 教师讲解文章整体结构和每段主题
4. 学生理解论证逻辑和段落功能

#### 第二课时（45分钟）- 分析句式
1. 进入**阶段二**
2. 先查看"原文视图"，整体阅读
3. 切换到"句式拆解视图"
4. 教师逐句分析主谓宾结构和复杂句式
5. 切换到"词汇扩展视图"
6. 讲解重点词汇和同义替换

#### 第三课时（40分钟）- 背诵练习
1. 进入**阶段三**
2. 点击"开始背诵"
3. 学生尝试背诵，点击查看SVO提示
4. 确认无误后，点击显示完整句子
5. 重复直至背完全文

#### 课后作业
- 学生在家使用系统自主练习
- 建议每天练习20-30分钟
- 一周内完成全文流利背诵

## 💡 使用技巧

### 教师使用建议
1. **第一阶段**用于课堂导入，时间控制在10分钟内
2. **第二阶段**是教学重点，建议分2-3次课完成
3. **第三阶段**适合课堂演示后，让学生课后自主练习
4. 可结合录屏功能，制作教学视频供学生复习

### 学生学习建议
1. 第一遍先理解意思，不要急于背诵
2. 第二遍关注句式结构和词汇
3. 第三遍开始背诵，利用SVO提示
4. 每天背诵3-5句，循序渐进
5. 背完一遍后，第二天复习巩固
